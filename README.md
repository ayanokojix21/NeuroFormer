# 🧠 NeuroFormer

**NeuroFormer** is a **modular Transformer architecture** built from scratch in PyTorch that supports:

- **Decoder-only** (like GPT) — for language modeling and chatbots  
- **Encoder-only** (like BERT) — for tasks like question answering  
- **Encoder-Decoder** (like T5) — for machine translation

This project aims to **demystify Transformer internals** by implementing all major components from scratch and training/fine-tuning them on real-world datasets — all deployable via a Streamlit app.
